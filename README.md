# Bank Management System in C

A console-based Bank Management System developed using the C programming language.  
This project demonstrates the use of structures, file handling, and modular programming to manage basic bank account operations.

---

## Features
- Create a new bank account
- View existing account details using account number
- Persistent data storage using binary files
- Menu-driven user interface
- Deposit and withdraw money from existing accounts

---

## Concepts Used
- C Structures
- File Handling (`fopen`, `fread`, `fwrite`)
- Binary files
- Functions and modular programming
- Menu-driven programming

---

## Project Structure
- bank.c : source code
- bank.exe : compiled program
- bank.dat : data file created during execution
---

## How to Run

1. Open terminal in the `src` folder  
2. Compile the program: gcc bank.c -o bank
3. Run the program: ./bank

---

## Notes
- This project was built as a learning exercise to understand file handling and structures in C.
- Additional features like deposit and withdrawal can be added in future versions.
